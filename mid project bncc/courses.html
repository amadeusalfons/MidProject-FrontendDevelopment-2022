Courses

Front End Development

15 meetings

This course teaches students how to build the visual and interactive elements of websites and web applications using HTML, CSS, and JavaScript.

Back End Development

20 meetings

This course covers the programming and database skills needed to build the server-side of web applications, including creating APIs, managing databases, and integrating with front-end interfaces.

Data Analytics

15 meetings

This course focuses on the process of analyzing and interpreting complex data sets using statistical methods, software tools, and machine learning algorithms to derive insights and make data-driven decisions.

Artificial Intelligence

10 meetings

This course covers the theory and application of machine learning and artificial intelligence techniques to solve problems such as natural language processing, image and speech recognition, and predictive modeling.
